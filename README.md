# Bandiit

Bandiit is a lightweight SaaS application to help bands organize their schedules and track attendance.
Built using NextJS, Convex, and Clerk, Bandiit offers responsive real-time updates on a polished UI.

## Features

- **Band Management**: Create and manage multiple bands. Invite members and assign roles (Admin/Member).
- **Event Scheduling**: Create events with details like location, time, and type (Gig, Rehearsal, etc.).
- **Smart RSVPs**: Real-time RSVP tracking for every member that updates as needed.
- **Event History**: Automatic archival of band activities for historical reference.
- **Mobile-first UI**: Responsive interface built with Tailwind CSS and Shadcn UI.

## Tech-Stack

- **Framework**: Next.js 16 (App Router)
- **Backend and Database**: Convex
- **Authentication**: Clerk
- **Styling**: CSS
- **UI Components**: Shadcn UI
- **Icon Kit**: Lucide React
